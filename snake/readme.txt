Makefile установит sfml и скомпилирует два object-файла
snake_user - управление клавиатурой
snake_bot - управление ботом
Каждый принимает 7 мараметров командной строки
width - ширина в клетках
height - высота в клетках
cell_cize - размер клетки в пикселях
model_freq - частота обновления модели
view_freq - частота обновления графического окна
controller_freq - частота, с которой контроллер читает события
difficulty - сложность (скорость роста частоты обновления модели)
Для удобства сделал два make-файла, которые запускают со стандартными параметрами
make -f run-bot - для бота
make -f run-user - для управления клавиатурой соответственно
